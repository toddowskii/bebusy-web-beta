# BeBusy 🐝

> **Social network for productive people**

BeBusy is a web application designed to connect people around productivity, goals, and getting things done.

🔗 **[THE LINK TO THE APP](https://bebusy-web.vercel.app)**

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | [Next.js](https://nextjs.org/) (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| Deployment | Vercel |

---

## Features

- 🔐 User authentication — sign up, log in, and password reset
- 📱 Responsive layout built for web and mobile
- ⚡ Fast page loads via Next.js App Router and server components

---

## Project Structure

```
bebusy-web-beta/
├── app/            # Next.js App Router pages and layouts
├── components/     # Reusable UI components
├── config/         # App-wide configuration
├── lib/            # Utility functions and shared logic
├── public/         # Static assets
├── types/          # TypeScript type definitions
└── middleware.ts   # Auth and routing middleware
```
